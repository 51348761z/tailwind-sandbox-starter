# Product Modal

A responsive product card with interactive elements built with Tailwind CSS.

## Features

- Responsive design for mobile and desktop
- Product image with hover scale effect
- Price comparison display
- Animated "Add to Cart" button
- Live stock indicator with ping animation
- Add to cart and wishlist buttons

## Technologies

- HTML5
- Tailwind CSS v4
